### Skill Mastery System

**6 Skill Categories:**
1. **Combat**: Physical fighting expertise
2. **Magic**: Magical abilities proficiency
3. **Stealth**: Sneaking and subterfuge
4. **Diplomacy**: Social interactions
5. **Crafting**: Creating items
6. **Exploration**: Finding secrets

**6 Mastery Levels:**
1. **Novice** (0-24 XP): Starting level, no bonuses
2. **Apprentice** (25-49 XP): +5 stat bonus
3. **Adept** (50-74 XP): +10 stat bonus
4. **Expert** (75-99 XP): +20 stat bonus
5. **Master** (100-149 XP): +35 stat bonus
6. **Grand Master** (150+ XP): +50 stat bonus

**How to Gain Skill XP:**
- Combat: +1 XP per enemy defeated
- Magic: +1 XP per spell cast
- Diplomacy: +1 XP per dialogue, +5 per companion recruited
- Crafting: +1 XP per item crafted
- Exploration: +2 XP per quest, +3 per location, +5 per secret
- Stealth: +1 XP per successful stealth action

### Statistics Tracking

The progression system tracks:
- Quests completed
- Enemies defeated
- Items crafted
- Locations discovered
- Secrets found
- Playtime (in hours)
- Deaths
- Companions recruited
- Dialogues completed
