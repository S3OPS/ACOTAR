## üóÇÔ∏è Archive Organization Policy

Files are moved to the archive when:
1. A new major version supersedes them
2. They represent completed historical phases
3. They are no longer actively referenced
4. They would clutter the root directory

Files are **kept in root** when:
1. They represent current/active documentation
2. They are frequently accessed by developers
3. They provide essential project information
4. They are part of the core documentation set
